## **What the game is about...**

**Ototime** is a 3D third-person parkour where you are timed on how fast you can complete the parkour!

## **Controls**

- W: Move Forward
- S: Move Backwards
- A: Move Left
- D: Move Right
- Spacebar: Jump/Start 


## **Features (current and future plans)**
- Oto model!
- Third-person gameplay
- Parkour Map and Skybox
- Sweet music!
